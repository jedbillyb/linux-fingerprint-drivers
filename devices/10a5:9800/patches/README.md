# Patches for FPC fpcmoh (10a5:9800)

No standalone patch files are mirrored here yet. The driver is distributed via
the `libfprint-fpcmoh-git` AUR package (see `../CREDITS`); build from there.

If you generate a patch series against a known libfprint base, drop the numbered
`.patch` files here and note the base commit. Remember the device also needs the
accompanying udev rules.
